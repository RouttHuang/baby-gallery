# 宝宝成长纪念册

一个美观漂亮、可交互的宝宝成长纪念册网站，适合编程新手使用，无需后端服务，无需API密钥。

## 功能特点

### 核心功能
- **欢迎词** - 温馨的欢迎页面，介绍网站的目的和意义
- **宝宝成长计时** - 可自定义生日的实时倒计时，显示天、小时、分钟、秒
- **宝宝相册** - 支持从本地目录上传和查看照片，网格布局，点击放大，滑动浏览
- **宝宝视频集锦** - 支持视频播放、点赞和评论
- **宝宝大事纪** - 时间线形式展示宝宝成长的重要里程碑
- **游客寄语留言** - 可提交和查看留言，记录亲友的祝福

### 技术特性
- **响应式设计** - 适配手机、平板、桌面等所有设备
- **现代UI设计** - 简约温馨的设计风格，清晰的视觉层次结构
- **流畅的交互体验** - 按钮悬停、卡片动画、页面滚动动画等微交互效果
- **高性能** - 图片懒加载、优化的页面加载速度
- **数据存储** - 使用浏览器内置的localStorage进行数据存储，无需后端服务
- **无API密钥** - 不需要任何API密钥，不会泄露敏感信息

## 技术栈

### 前端
- **React 18** - 现代化的前端框架
- **Vite** - 快速的构建工具
- **TailwindCSS 3.0+** - 实用优先的CSS框架
- **LightGallery** - 强大的图片/视频画廊库

### 数据存储
- **localStorage** - 浏览器内置的本地存储，用于存储照片、视频、点赞和评论数据

## 快速开始

### 1. 安装依赖

```bash
# 进入项目目录
cd simple-baby-gallery

# 安装依赖
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看网站

### 3. 构建生产版本

```bash
npm run build
```

构建完成后，生产文件将生成在 `dist` 目录中

## 如何部署到 Vercel

### 步骤 1: 创建 GitHub 仓库

1. 访问 [GitHub](https://github.com/) 并登录
2. 点击右上角的 "+" 按钮，选择 "New repository"
3. 填写仓库名称（例如：`baby-gallery`）
4. 选择 "Public" 或 "Private"（建议选择 "Public"）
5. 点击 "Create repository"

### 步骤 2: 将项目上传到 GitHub

1. 打开命令行工具，进入项目目录

```bash
cd simple-baby-gallery
```

2. 初始化 Git 仓库

```bash
git init
```

3. 关联 GitHub 仓库

```bash
git remote add origin https://github.com/你的GitHub用户名/你的仓库名称.git
```

4. 添加所有文件

```bash
git add .
```

5. 提交代码

```bash
git commit -m "Initial commit"
```

6. 推送到 GitHub

```bash
git push -u origin main
```

### 步骤 3: 部署到 Vercel

1. 访问 [Vercel](https://vercel.com/) 并登录
2. 点击 "Add new project"
3. 选择 "Import GitHub Repository"
4. 找到你的仓库并点击 "Import"
5. 配置项目：
   - Framework Preset: 选择 "Vite"
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - 不需要添加任何环境变量，因为我们使用 localStorage 存储数据
6. 点击 "Deploy"
7. 等待部署完成，获取可访问的 URL

## 信息安全说明

### 为什么这个项目不会泄露敏感信息？

1. **无API密钥** - 项目不使用任何第三方API服务，因此不需要API密钥
2. **数据存储** - 所有数据都存储在用户浏览器的localStorage中，不会上传到任何服务器
3. **纯前端项目** - 这是一个纯前端项目，没有后端代码，不会有服务器端的安全问题
4. **本地文件处理** - 照片和视频仅在浏览器中处理，不会上传到任何服务器

### 如何确保部署过程中不会泄露敏感信息？

1. **不添加环境变量** - 我们的项目不需要任何环境变量，因此在部署到Vercel时不需要填写任何敏感信息
2. **检查文件** - 确保你没有在项目中添加任何包含敏感信息的文件
3. **使用.gitignore** - 项目中已经包含了.gitignore文件，会自动忽略node_modules、.env等不必要的文件

## 使用说明

### 1. 设置宝宝生日

1. 访问 "宝宝成长计时" 页面
2. 点击日期选择器，选择宝宝的实际生日
3. 系统会自动保存生日信息，下次访问时会记住你的设置

### 2. 上传照片

1. 访问 "宝宝相册" 页面
2. 点击 "选择本地照片" 按钮
3. 从本地目录选择一张或多张照片
4. 照片会自动显示在相册中，并保存到浏览器的localStorage中

### 3. 上传视频

1. 访问 "视频集锦" 页面
2. 点击 "选择本地视频" 按钮
3. 从本地目录选择一个或多个视频
4. 视频会自动显示在视频列表中，并保存到浏览器的localStorage中

### 4. 点赞和评论

1. 访问 "宝宝相册" 或 "视频集锦" 页面
2. 找到你喜欢的照片或视频
3. 点击心形图标进行点赞
4. 在评论框中输入评论，点击 "发送" 按钮提交评论

### 5. 查看留言

1. 访问 "游客寄语" 页面
2. 可以查看其他访客的留言
3. 你也可以在留言表单中提交自己的祝福

## 注意事项

1. **数据存储** - 所有数据都存储在浏览器的localStorage中，如果清除浏览器数据，所有照片、视频和评论都会丢失
2. **照片和视频大小** - 由于数据存储在浏览器中，建议不要上传过大的照片和视频，以免影响浏览器性能
3. **跨浏览器兼容** - 数据仅存储在当前浏览器中，无法在不同浏览器之间共享
4. **隐私保护** - 虽然数据存储在本地，但如果你的电脑被他人使用，他们可能会看到你的照片和视频
